# 更新日志
## 版本说明
Openjob 版本号命名规则为：v大版本.功能版本.修复版本。
:::tip 例如
v1.0.1 是 v1.0.0 之后的Bug修复版本 v1.1.0 是 v1.0.0 之后的功能版本
:::

Openjob 将按月发布功能版本、按年度发布大版本。
